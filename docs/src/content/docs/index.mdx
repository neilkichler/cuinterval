CuInterval is a CUDA [interval arithmetic](https://en.wikipedia.org/wiki/Interval_arithmetic) library. It includes operator-overloading based device kernels for all fundamental and set-based interval operations of the [IEEE Standard for Interval Arithmetic](https://doi.org/10.1109/IEEESTD.2015.7140721).
